## 타임라인 SNS 서비스 (chanstagram)

**인스타그램과 유사한 기능을 가진 웹 어플리케이션을 구현하여 Spring Framework 학습**

<br>

- **기능**
  - 로그인, 회원가입
  - 게시물 upload
  - 팔로우 (구독)
  - 팔로우 할 사용자 추천
  - 타임라인
  - 좋아요, 댓글 - 구현 예정

<br>

- **기술 스택**
  + JAVA
  + Spring boot
  + Spring Security
  + Spring Data JPA
  + RDBMS (h2)
  + Thymeleaf
  + HTML, CSS, Javascript, jQuery, Bootstrap

<br>

+ **개발 환경**
  + macOS Catalina (10.15.4)
  + IntelliJ IDEA IDE
